Representing safe areas in Indian metro cities as defined by proximity to Emergency Services

Made using Folium

February 2022

Details:

Emergency Services considered are Police Stations, Fire Stations and Hospitals
Proximity limits (in mins) are taken as inputs from the user
Coordinates of Emergency Services and their individual isochrone maps are gathered using API calls

Data Source:

Open Route Service
Open Street Map

Map assessing the metro accessibility in Bangalore

Made using QGIS

January 2022

Details:

A buffer zone of 1Km radius is created around metro stations by temporarily reprojecting the metro data layer to the Datum: EPSG:32643 - WGS 84 UTM

Data Source:

Population count: wordpop.org (https://www.worldpop.org/geodata/summary?id=49992)
Metro stations location: Overpass API (API accessed using Quick OSM in Qgis)
Bangalore map: spatialthoughts.com (https://docs.google.com/forms/d/e/1FAIpQLSdwB5I9PdheF8V4yOSeoGSkmEdBZoe4R9CyFQVzlFH-pTl6FQ/viewform)
